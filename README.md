# AGGlory Panel

  ## Railway Deploy Karne Ka Tarika

  1. Is folder ko GitHub pe push karo
  2. Railway.app pe jaao → New Project → Deploy from GitHub
  3. PORT environment variable Railway automatically set karta hai
  4. Deploy!

  ## Local Run Karne Ke Liye
  ```bash
  npm install
  npm run build
  npm start
  ```
